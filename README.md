# React + Vite + TailwindCSS v4

start: npm run dev

## Folder Structure:

- page:main UI
- component: contains reusable components
- feature: structured into feature-based folders, each handling its own UI and core logic.
- utils: dataMock: mock data simulating API responses/ enum: predefined constants
